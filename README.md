
  # Wedding website design

  This is a beautiful, modern wedding invitation website with enhanced features including FAQ, timeline, accommodations, and improved accessibility.

  ## 🎉 New Features

  - **FAQ Section**: Comprehensive answers to common questions about dress code, dietary needs, parking, and more
  - **Event Timeline**: Detailed schedule of ceremony, cocktail hour, reception, dancing, and send-off
  - **Accommodations**: Hotel recommendations with special rates and booking information
  - **Improved RSVP Form**: Real-time validation, error messages, loading states, and dietary restrictions field
  - **Navigation Menu**: Smooth scrolling navigation to jump between sections
  - **Gallery Enhancements**: Keyboard navigation (arrow keys), ESC to close, image counter, lazy loading
  - **Accessibility**: Skip links, ARIA labels, keyboard navigation, focus management
  - **SEO Optimized**: Meta tags, Open Graph tags, structured data for search engines
  - **PWA Ready**: Manifest file for "Add to Home Screen" capability on mobile devices

  ## 🎨 Features

  - Responsive design for all devices
  - Smooth scroll animations with Motion
  - Interactive photo gallery with modal viewer
  - Countdown timer to wedding day
  - RSVP form with validation
  - Event details and venue information
  - Mobile-friendly navigation menu

  ## 🚀 Running the code

  Run `npm install` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to create a production build.

  ## 📝 Customization

  All wedding-specific constants (dates, venue, colors, etc.) can be found in `src/constants.ts` for easy customization:
  - Wedding date and time
  - Venue information
  - Color palette
  - Timeline schedule
  - Hotel recommendations
  - FAQ items

  ## ♿ Accessibility Features

  - Keyboard navigation throughout
  - Screen reader support with ARIA labels
  - Skip to main content link
  - Focus management in modals
  - Color contrast compliance
  - Proper semantic HTML

  ## 🎯 Browser Support

  Modern browsers including Chrome, Firefox, Safari, and Edge.

  ## 📱 PWA Features

  This website can be installed as a Progressive Web App on mobile devices, allowing guests to save it to their home screen for easy access.
